#### :ballot_box_with_check:使用教程

使用者无需关心免杀实现,只需要在Windows x64位系统运行即可

命令行输入:ByPassBehinder.exe

![01](\image\01.jpg)

目前支持格式为常见WebShell全版本格式:ASP,ASP.NET,PHP,JSP,JSPX

输入对应的编号即可生成,默认密码为**Tas9er**
